# Learning Qiskit

This repository contains my notes and code while learning Qiskit.

## Using Poetry As Package Manager

- [Poetry](https://python-poetry.org/docs/#installation)

## venv

```bash
python3 -m venv venv
```

- Activate venv

```bash
source .venv/bin/activate
```

or

```bash
make activate
```

## Install dependencies

```bash
poetry install
```

## Add Packages

```bash
poetry add numpy
```
